mt2026

Este repositório é uma experiência em realidade aumentada usando AR.js e A-Frame.

Para publicar como site no GitHub Pages, basta ativar a página do GitHub e usar este arquivo index.html como raiz.

Passos para habilitar o site

  1. Faça commit e push das alterações para o branch main.
  2. Acesse Configurações de Pages do repositório.
  3. Em "Branch", selecione main e em "Pasta" selecione / (root).
  4. Salve e aguarde alguns minutos até o site ficar disponível.

Depois, o site estará disponível em:

https://samuellps00.github.io/mt2026/

Observações

O README atual contém código de exemplo para experiências de AR, mas o site precisa de arquivos adicionais (como imagens e modelos 3D) para funcionar completamente.

Se quiser, posso também ajudar a criar uma versão completa do site com as cenas de AR quando você adicionar os arquivos de recurso.
