mt2026 — Experiência AR

Importante: permita o uso da câmera no navegador. Aponte para o marcador hiro para ver a cena.

Este site converteu o conteúdo do README.md em uma página AR funcional. Você pode alternar entre duas experiências:

  • Imagem + marcador: exibe uma bandeira 2D sobre o marcador hiro.
  • Modelo 3D + marcador: exibe um modelo glTF sobre o mesmo marcador.

Se quiser, posso ajustar ainda mais para usar arquivos locais como pattern-marcador.patt, bandeira.png ou macaca.glb.

Abra esta página no navegador com câmera e aponte para o marcador hiro. Você pode encontrar o marcador aqui:

https://raw.githubusercontent.com/AR-js-org/AR.js/master/three.js/examples/marker-training/examples/pattern-files/pattern-hiro.patt

Se estiver usando GitHub Pages, o endereço será:

https://samuellps00.github.io/mt2026/
